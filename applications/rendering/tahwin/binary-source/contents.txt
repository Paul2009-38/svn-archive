this directory contains binaries included with the installer
extract files flat into mentioned subdirectories

\fonts\
  copy *.ttf into this folder from dejavu and unifont
  source http://sourceforge.net/projects/dejavu/files/dejavu/dejavu-fonts-ttf-2.30.zip
  source (http://unifoundry.com/unifont.html) http://unifoundry.com/unifont-5.1.20080907.zip
  rename unifont to unifont.ttf
 
\pngnq\
  source http://sourceforge.net/projects/pngnq/files/pngnq/0.5/pngnq-0.5-i386-win.zip/download
  I included two dlls (libpng13.dll, zlib.dll) that are missing in the distribution but required and hard to find

\snv\
  source http://subversion.tigris.org/files/documents/15/47914/svn-win32-1.6.6.zip
  cmd-line client for SVN

\batik\
  source http://www.apache.org/dist/xmlgraphics/batik/batik-1.7.zip
  source xercesImpl.jar from http://www.apache.org/dist/xerces/j/Xerces-J-bin.2.11.0.zip
  source NOTICE (as NOTICE-XERCES) from http://www.apache.org/dist/xerces/j/Xerces-J-bin.2.11.0.zip
  remove subdir doc and samples
  Apache Batik and Xerces Java

\xmlstarlet\
  source http://sourceforge.net/projects/xmlstar/files/xmlstarlet/1.0.3/xmlstarlet-1.0.3-win32.zip/download
  XML Starlet
  
\optipng\
  source http://prdownloads.sourceforge.net/optipng/optipng-0.6.4-exe.zip?download
  
\pngcrush\
  source http://sourceforge.net/projects/pmt/files/pngcrush-executables/1.7.9/pngcrush-1.7.9-win32.zip/download
    
\zip\
  source http://sourceforge.net/projects/sevenzip/files/7-Zip/9.20/7za920.zip/download
