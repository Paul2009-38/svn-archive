OSMHealthCheck is a small bot to monitor the health status of various OSM services, by
pinging them at regular intervals.


http://wiki.openstreetmap.org/wiki/Osm-healthcheck

=== External sources ===

OSMHealthCheck includes code from:

- pircbot, a java library for IRC comunications.

It is found at http://www.jibble.org/pircbot.php
and licensed under the GPL.

- Osmosis
for Date parsing of the minutely diff status files.
